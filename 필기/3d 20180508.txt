정점 쉐이더 - 정점에 대한 연산
- 입력, 조립 -> 정점쉐이더 순
- 각 정점에 대한 연산 수행
- 하나의 정점에 한번 호출되며 반드시 하나의 출력 정점(연산의 결과)을 생성
- SV_VertexID	: 정점쉐이더에서 사용할 수 있음
  SV_InstanceID

픽셀 쉐이더 - 픽셀에 대한 연산
- 레스터 라이저를 거쳐 온 픽셀들에 대해서 픽셀쉐이더 함수를 호출함 
- 연산의 결과로 색깔을 반환

출력 - 병합 단계
- 깊이, 스텐실 검사
- 블렌딩

시멘틱
- 쉐이더의 입력 또는 출력 파라메터 변수에 부착되는 문자열
- 변수의 사용 의도를 나타내는 정보를 컴파일러에게 전달
- 입력과 출력을 연결시켜준다
- SV로 시작하는 시멘틱들은 그 용도에 맞게 써야함 정의가 되어있음(시스템 값 시멘틱)
- 숫자는 0~7까지만 가능

강의 자료를 출력해서 들고다녀라

정점 쉐이더 준비하기
1) HLSL를 사용하여 함수를 만든다
2) 컴파일을 함(D3DCompileFromFile()) -> 바이트 코드들을 연결
3) 파이프라인 상태의 VS 단계에 설정한다
(D3D12_GRAPHICS_PIPELINE_STATE_DESC::VS, ID3D12Device::CreateGraphicsPipelineState())
